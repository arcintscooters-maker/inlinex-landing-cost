# InlineX Landing Cost Calculator (Python)

## Setup

1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Run the app:
   ```
   python app.py
   ```

3. Open browser at: **http://localhost:5000**

## Usage

1. Enter your Anthropic API key (saved in browser localStorage)
2. Upload supplier invoice (PDF or Excel .xlsx)
3. Enter SGD amount paid for invoice
4. Enter SGD shipping fee
5. Click Calculate — AI reads invoice and calculates landed cost
6. Export CSV for Shopify PO import

## Requirements
- Python 3.9+
- pip
